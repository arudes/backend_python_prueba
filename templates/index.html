Detalles del proyecto


El proyecto se ejecuta en 127.0.0.1:4000 o localhost:4000

Las rutas proporcionadas de ben de ser escritas en la barra del navegador

Con respecto al servicio de obtener los 20 mejores y ultimos podcast se debe de establecer la ruta donde se creara el archivo dentro del proyecto

Se utilizo datos JSON como fuente

  • Servicio para buscar podcast por medio de nombre del artista (artistName)
  • ruta: /podcast/nombre del artista
  • Algunos nombres con los que se puede probar:
    • iHeartRadio
    • Joe Rogan
    • Armchair Umbrella
  • Para el servicio de guardar los 20 mejores en un JSON
  • ruta: /best20 - Crea un archivo dentro de la carpeta del proyecto
  • Para reemplazar los 20 podcast principales por los 20 inferiores en un archivo JSON
  • ruta: /last20 - Reemplaza el contenido del archivo creado de los 20 mejores podcast
  • Para eliminar podcast por medio del nombre del artista (artistName)
  • ruta: /podcastdel/nombre del artista >

    • Usando una base de datos SQLite3 llamada: podcast.db


  • Servicio que lista los veinte podcast de la base de datos
  • ruta: /podcastdb - Se listan en formato JSON
